Detailed Description

This is a utility class to incrementally calculate average, variance and standard deviation. It's based on an algorithm devised by Knuth.

Please keep in mind, that there might be edge cases where the below algorithm might produce a loss of precision.

See below link for more detail:

http://en.wikipedia.org/wiki/Algorithms_for_calculating_variance#Incremental_algorithm
